ActiveRecord: find vs find_by

When performing ActiveRecord find and find_by there is an important difference that I ran into this month. A simple find will only search for the id from the model and will return an exception if it doesn't exist. While a find_by_[field_name] will search the specified field, but will return nil if it doesn't exist. Helpful to prevent exceptions when you aren't sure if a record exists or not.
