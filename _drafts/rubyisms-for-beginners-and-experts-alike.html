1. Default variable value

Also known as memoization, this line of code sets a variable equal to the value on the right side unless the variable is already defined. Useful for code that stays the same over multiple loops or expensive active record queries.

{% highlight ruby %} my_age ||= 'default' {% endhighlight %}

You can even memoize an entire block using being / end.

{% highlight ruby %} memoized ||= begin # A block of code that is reused frequently end {% endhighlight %}

While a useful feature, be careful when over-memoizing as caching issues can become an issue.

2. Post conditionals

my_age = 23 if status == 200 company_name = Company.first.name unless Company.size == 0

3. Guard clauses

def search return unless results.present? results.status = :success end

4. Array Push

age_list = [23, 40, 28, 27, 32] age_list << 54 puts age_list

5. Percent String Arrays

%w[Gandalf Middle-earth The\ Necromancer] #=> ["Gandalf", "Middle-earth", "The Necromancer"]

6. Splat arguments

def serious_method(first_param, *extra_args)

7. String Interpolation

my_name = 'Frodo Baggins' puts "Welcome to Rivendell, #{my_name}"

8. Blocks

variable.each do |x| puts x end VS. variable.each { |x| puts x }

9. Multiple statements on single line

(0..5).inject { |sum, n| break sum if sum > n; sum + n }
