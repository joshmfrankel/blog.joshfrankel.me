Going from one programming language to another, there is a lot of knowledge that is easy to bring along. Theories, patterns, and general programming standards are pretty much universal across languages. However, syntax and process between two languages rarely looks the same.

Since I come from a background in PHP development, I find it easy to compare that knowledge with how rails accomplishes the same feats.

Rails comes with awesome command line support

rails g rails s rails c rails scaffold

Everything is an object!

Methods

Everything is an Object
