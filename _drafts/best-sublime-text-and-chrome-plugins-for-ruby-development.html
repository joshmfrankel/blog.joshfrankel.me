As part of the unofficial, yet soon to become official compilation series on getting started with Rails, this article will focus on another aspect of development: useful text editor and browser plugins. These are extensions I have used at some point and found to be highly effective and increasing productivity. This guide assumes that you are using Chrome and Sublime Text. Enjoy!

Chrome

Rails Panel

I just recently discovered this plugin for Chrome and I can say without a doubt that it is easily one of the best methods for understanding what Rails is doing in the background. This plugin lays out everything for you that is going on behind the scenes.

Here are a just few of its features:

  • Displays all ActiveRecord queries
  • Lists all views being rendered with duration
  • Catches any debug logs from within the application
  • Shows all page errors
  • Elaborates on the breakdown of the url params for the page

This plugin has been extremely helpful for me in understanding Rails application structure as well as tracking down pesky errors quicker.

To setup using Rails panel you need to do the following:

  1. Install the Rails Panel extension in chrome here
  2. Add the gem to your gemfile gem 'meta_request'. (Preferably within a :development block)
  3. Run bundle in your app's directory to reload the gems
  4. Restart your rails server (rails s) and it should start catching application events
